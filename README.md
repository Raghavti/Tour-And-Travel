## 🚀 Getting Started

### 📌 Prerequisites

Make sure you have installed:

* Node.js
* npm or yarn

### 📥 Installation

```bash
git clone https://github.com/your-username/tour-travel-app.git
cd tour-travel-app
npm install
```

### ▶️ Run the Project

```bash
npm start
```

or

```bash
npm run dev
```

Now open your browser and visit:
`http://localhost:3000`
